# House Price Prediction Based on Location
## Project Overview
This project predicts house prices based on geographical location and other housing features using Machine Learning techniques.
## Objective
To build a machine learning model that predicts house prices accurately based on location-related attributes.
## Dataset
The California Housing Dataset is used in this project.
Features:
- Longitude
- Latitude
- Housing Median Age
- Total Rooms
- Total Bedrooms
- Population
- Households
- Median Income
Target:
- Median House Value
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
## Machine Learning Algorithm
- Linear Regression
## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Prediction
## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
## Conclusion
The model predicts house prices based on location and housing characteristics with good accuracy.
